El programa debe ser ejecutado desde la consola pues existe 
un error en el codigo de la de la cual se nos dieron los videos,
el cual es el path, el escribio el path de su computadora pero 
varia segun la computadora donde sea ejecutado. 
Se utilizo VSC como entorno en el que se edito, cambio e hizo modificaciones,
como añadir un diseño distinto de UFO y cambiar los controles a W, A y D para moverse
y espacio para disparar.

Se puede ejecutar con la extension de java de VSC, dando
click derecho a la clase Window.java y escogiendo la opcion
de run java y tambien en consola utilizando este comando:
java -cp "bin;lib/json-20180813.jar;res" main.Window
